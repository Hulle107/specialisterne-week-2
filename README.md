## 🚥 Start

### 🔍 Afhængigheder

- Node.js 18.18 eller nyere. ([link](https://nodejs.org))
- Permit.io ([link](https://permit.io))
- Internetforbindelse

### 💾 Installering

```shell
# Downloader & installere alle nødvendige filer
npm install -y
```

```shell
# Bygger applikationen
npm run build
```

### 🏃 Kør program

```shell
# Starter applikationen
npm run start
```
